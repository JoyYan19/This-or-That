# Soccer Goal Decision Tool
This app helps determine whether a controversial goal in a soccer match should count or not. By answering questions about the situation (offside, handball, fouls, goalkeeper interference, and goal line), the app analyzes the evidence and renders a verdict on whether the goal is valid.

**Disclaimer:** This tool is for entertainment and educational purposes only and should not be used for official match decisions.

## Questions Covered:
1. **Offside** - Was the attacker in an offside position?
2. **Handball** - Did the player use their hand/arm to score?
3. **Fouls** - Was there a foul before the shot?
4. **Goalkeeper Interference** - Was there illegal interference?
5. **Goal Line** - Did the ball completely cross the goal line?

## Technologies:
- HTML5
- CSS (W3.CSS framework with flat colors)
- JavaScript

The app uses a scoring system to determine the verdict - if your answers suggest the goal is valid in 60% or more of the reviewed factors, it counts!
